# Client Onboarding Checklist — 14-Day Delivery

The point of this doc: **fulfillment stops being the bottleneck on growth.** Every client follows this exact path. No custom-tailoring. The second you start reinventing onboarding for each client, you cap yourself at ~5 clients.

---

## Day 0 — Deal closed

**Trigger:** proposal signed + deposit paid.

- [ ] Stripe confirmation received and logged in pipeline (move to "Won")
- [ ] Send welcome email (template below) within 15 minutes of payment
- [ ] Kickoff call booked within 48 hours
- [ ] Client added to Notion client database with: package, launch date, deliverables, contact info
- [ ] Create shared Google Drive folder: `{{Client Name}} — {{Package}} — Launch {{Date}}`
- [ ] Add launch date to your calendar as a HARD deadline

## Day 1 — Kickoff call (30 min)

- [ ] Confirm deliverables in writing (screen-share the proposal)
- [ ] Collect: logins (hosting, domain, Google Business Profile, existing site CMS), brand assets (logo, colors, photos), content (about copy, service descriptions, team bios)
- [ ] Walk through the 14-day timeline
- [ ] Set expectation: 2 rounds of revisions, each with 48h turnaround from client
- [ ] Send the **Asset Request Form** (Google Form) immediately after call
- [ ] Set Day 3 deadline for client to submit all assets

## Day 2–3 — Asset collection + design

- [ ] Asset form received — if not, follow up at 5pm Day 2
- [ ] Draft sitemap + wireframes for 7 pages
- [ ] Homepage design concept delivered for approval by end of Day 3

## Day 4–10 — Build

- [ ] Build all pages (Home → About → Services → 3 service pages → Contact)
- [ ] Mobile optimization pass
- [ ] Page-speed pass (target <2s, Google PageSpeed 90+)
- [ ] Install LeadCatch form on Contact + Home
- [ ] Configure FAQ Chatbot (25 Q&A from Asset Form intake)
- [ ] Connect booking integration
- [ ] On-page SEO: titles, meta, schema markup for {{niche}}
- [ ] Call tracking number installed
- [ ] Form submission notifications routed to client's preferred email/SMS

## Day 11–12 — Revisions round 1

- [ ] Send preview link with 2-min Loom walkthrough
- [ ] Client has 48h to submit consolidated revision list (one document, not scattered messages)
- [ ] Execute revisions

## Day 13 — Revisions round 2 + QA

- [ ] Second revision pass
- [ ] Full QA checklist (below)
- [ ] Client final approval

## Day 14 — Launch

- [ ] DNS cutover (schedule for off-hours for the business)
- [ ] Test forms, bot, phone tracking, booking from 3 devices
- [ ] Google Business Profile updated with new URL
- [ ] Google Search Console + Analytics installed
- [ ] Send launch email with: live URL, Loom of how to use the admin dashboard, the **14-day money-back guarantee** terms in writing (starts on launch day), referral ask pre-queued for Day 30
- [ ] Monthly billing initiated in Stripe (first charge 30 days out)
- [ ] Client moved to "Active — Month 1" in pipeline

---

## QA Checklist (Day 13)

Run this on 3 devices: desktop, iPhone, Android.

- [ ] All pages load <2s on 4G
- [ ] All forms submit and notify the right email
- [ ] Bot answers 5 test questions correctly + books a test appointment
- [ ] Phone number on every page click-to-call on mobile
- [ ] All outbound links work
- [ ] 404 page exists and has a CTA back to home
- [ ] Favicon, meta titles, OG images all correct
- [ ] Google Analytics firing on every page
- [ ] Call tracking number ringing the right phone
- [ ] SSL certificate active
- [ ] Sitemap submitted to Google Search Console

---

## Day 30 — Client Success Checkpoint

- [ ] Pull analytics: visitors, form submissions, bot conversations, booked calls
- [ ] Send Loom review (5 min) with 2 wins + 1 recommendation
- [ ] Send first upsell email from `/outreach/upsell-sequences.md` if on Starter
- [ ] Send `/outreach/referral-ask.md` if they got at least 1 new lead
- [ ] Update pipeline with upsell intent + MRR status

## Day 60 — Case Study Checkpoint

- [ ] If they've had 3+ leads or a closed customer: start case study using `/assets/case-study-templates.md`
- [ ] Offer to feature them on LinkedIn + email list (use `/assets/client-spotlight-template.md`)
- [ ] Second upsell nudge if applicable

## Day 90 — Retention Checkpoint

- [ ] Quarterly Loom review with analytics + next-quarter recommendations
- [ ] Annual billing pitch: "Prepay 10 months, get 2 free"
- [ ] Renewal signaled → lock in another 12 months

## Capacity math

- 1 client takes ~12 focused hours over 14 days
- You have ~20 delivery hours/week = 2.8 launches per 2 weeks = ~6/month max before you hire
- **At 8+ active builds in parallel, you must hire a contractor.** Budget $800–1,200/client for a subcontracted builder.
- The moment delivery slips to Day 16+ consistently, growth stops. Onboarding is the throttle.
