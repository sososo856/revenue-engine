# Revenue Leak Audit — Template for Every Offer Funnel
# Find where prospects drop off and fix it
# Updated: April 2026

---

## How to Use This

Run this audit on each of your offer funnels once per month. It takes 30-60 minutes. The goal is to identify exactly where prospects drop off and fix the leakiest point first.

---

## Funnel Audit Template

### Funnel Name: [e.g., "Cold Audit → Growth Package"]

**Date of audit:** [Date]
**Period reviewed:** [Last 30/60/90 days]

### Stage 1: Awareness → Interest

**Channel:** [How prospects first encounter you — LinkedIn, cold email, referral, etc.]

| Metric | Number | Rate |
|--------|--------|------|
| People reached (impressions/emails sent) | | |
| People who engaged (opened/clicked/replied) | | % |

**Leak analysis:**
- Is the open/click/engagement rate below benchmark? (Email: 40%+ open, 5%+ reply. LinkedIn: 5%+ engagement)
- If yes, what's the likely cause?
  - [ ] Weak subject line / hook
  - [ ] Wrong audience / targeting
  - [ ] Bad timing (day/time of send)
  - [ ] Content not resonating
  - [ ] Deliverability issues (email landing in spam)

**Fix:** [Specific action — e.g., "A/B test 3 new subject lines" or "Refine targeting to only businesses with 10+ Google reviews"]

### Stage 2: Interest → Conversation

**Touchpoint:** [What happens after they engage — they reply, click a link, visit your site, etc.]

| Metric | Number | Rate |
|--------|--------|------|
| Engaged prospects | | |
| Conversations started (replies, DMs, calls) | | % |

**Leak analysis:**
- Are engaged people not starting conversations?
- If yes, what's the likely cause?
  - [ ] CTA is unclear or missing
  - [ ] Too much friction to respond (long form, complex next step)
  - [ ] No urgency or reason to act now
  - [ ] Landing page / audit video not compelling enough
  - [ ] No follow-up after initial engagement

**Fix:** [Specific action — e.g., "Add a direct calendar link to every Loom email" or "Send follow-up 48 hours after Loom delivery"]

### Stage 3: Conversation → Call Booked

**Touchpoint:** [How conversations become calls — calendar link, scheduling over DM, etc.]

| Metric | Number | Rate |
|--------|--------|------|
| Conversations started | | |
| Calls booked | | % |

**Leak analysis:**
- Are people replying but not booking calls?
- If yes, what's the likely cause?
  - [ ] Calendar link not provided or buried
  - [ ] Too many steps to book (back-and-forth scheduling)
  - [ ] No clear value proposition for the call
  - [ ] Delay in your response (they lost interest)
  - [ ] They got what they needed from the free audit

**Fix:** [Specific action — e.g., "Include calendar link in every response" or "Offer to call them instead of asking them to book"]

---

### Stage 4: Call Booked → Call Completed

**Touchpoint:** [Confirmation, reminders, show-up rate]

| Metric | Number | Rate |
|--------|--------|------|
| Calls booked | | |
| Calls completed (showed up) | | % |

**Leak analysis:**
- Are booked calls not happening (no-shows)?
- If yes, what's the likely cause?
  - [ ] No confirmation email/text
  - [ ] No reminder 24 hours and 1 hour before
  - [ ] Too long between booking and call date
  - [ ] Prospect lost enthusiasm / urgency faded
  - [ ] Calendar invite not sent properly

**Fix:** [Specific action — e.g., "Add SMS reminder via Twilio 1 hour before call" or "Book calls within 48 hours of interest"]

### Stage 5: Call Completed → Proposal Sent

**Touchpoint:** [The sales call itself]

| Metric | Number | Rate |
|--------|--------|------|
| Calls completed | | |
| Proposals sent | | % |

**Leak analysis:**
- Are calls not converting to proposals?
- If yes, what's the likely cause?
  - [ ] Bad fit discovered on call (not a real prospect)
  - [ ] Sales call structure needs improvement
  - [ ] Not enough social proof / case studies shared
  - [ ] Pricing not presented clearly
  - [ ] Objections not handled effectively
  - [ ] No urgency created on the call

**Fix:** [Specific action — e.g., "Practice the sales call framework" or "Add a new case study for their specific industry"]

---

### Stage 6: Proposal Sent → Deal Won

**Touchpoint:** [Proposal delivery and follow-up]

| Metric | Number | Rate |
|--------|--------|------|
| Proposals sent | | |
| Deals won | | % |

**Leak analysis:**
- Are proposals not converting to wins?
- If yes, what's the likely cause?
  - [ ] Proposal sent too late (enthusiasm cooled)
  - [ ] Pricing surprise (different from what was discussed on call)
  - [ ] No follow-up after proposal sent
  - [ ] Competitor swooped in
  - [ ] Decision maker not on the original call
  - [ ] Guarantee not prominently featured

**Fix:** [Specific action — e.g., "Send proposal within 2 hours of call" or "Add results guarantee to every proposal"]

### Stage 7: Won → Retained (for monthly clients)

**Touchpoint:** [Ongoing delivery and relationship]

| Metric | Number | Rate |
|--------|--------|------|

| Clients retained at 6 months | | % |

- Are clients churning?

  - [ ] First 30 days (expectations not met)
  - [ ] Month 2-3 (novelty wore off, not seeing results)

## Quick Funnel Health Score

After completing the audit, score each stage:

| Stage | Conversion Rate | Benchmark | Status |
|-------|----------------|-----------|--------|
| Awareness → Interest | % | 5-10% | Green/Yellow/Red |
| Interest → Conversation | % | 20-30% | Green/Yellow/Red |
| Conversation → Call | % | 50-70% | Green/Yellow/Red |
| Call → Proposal | % | 70-80% | Green/Yellow/Red |
| Proposal → Won | % | 40-60% | Green/Yellow/Red |
| Won → Retained (3mo) | % | 85-95% | Green/Yellow/Red |

**Focus on the reddest stage first.** That's your biggest leak.

---

## Monthly Review Cadence

- First Monday of every month: run this audit on your primary funnel
- Compare to last month's numbers
- Identify the biggest leak
- Implement one specific fix
- Review the impact next month

The goal isn't perfection at every stage — it's finding and fixing the worst leak, one at a time.
